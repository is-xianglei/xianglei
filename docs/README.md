## 博客搭建
- 本仓库是基于 docsify 构建的，想要的同款的铁子可以fork我的仓库，配置基础的信息就可以拥有一个这样的博客了！

**为什么不用 CSDN、掘金、简书？**

当然可以用(`扩大影响力`)，只不过这些都不是原稿存放平台，你的内容输出需要被随心所欲的维护和积累，以及将来需要的时候还可以放置到任何一个其他平台。而你写在 GitHub 仓库，就是最原始的那一份，其他的都是备份传播，这样更有益于你来沉淀、积累和分享。
